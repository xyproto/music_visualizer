# Music Visualizer

Visualize music and audio. Compiles on Arch Linux using [cxx](https://github.com/xyproto/cxx).

![Example visualization](img/muviz.gif)

## Build instructions

    cxx

## Package instructions

    cxx pkg

## Dependencies

    ffts glfw libpulse rapidjson
